    Functionality: Stubs typically perform minimal or no actions. They might print a message indicating they were called, or they might return a default value.
    Benefits:
        Allow testing of the main program flow and how functions interact with each other.
        Help programmers focus on other parts of the code without getting sidetracked by incomplete functions.

Overloaded Functions

    Concept: Function overloading allows you to have multiple functions with the same name but different parameter lists. The compiler determines which function to call based on the number and types of arguments provided at the call site.
    Purpose: Overloading enables functions to handle different data types or variations of the same operation.
    Example:
