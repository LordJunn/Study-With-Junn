The statement cout << setprecision(2) << showpoint; 
sets the output format for decimal numbers. 

The setprecision(2) part of the statement tells the output stream to display two decimal places for floating-point numbers. 

The showpoint part of the statement tells the output stream to always display the decimal point for floating-point numbers, 
even if the fractional part is zero.

If you change the instruction to cout << setprecision(2) << fixed << showpoint;, 
the program will output the total bill with two decimal places, 
even if the fractional part is zero. 

This is because the fixed part of the statement tells the output stream to display floating-point numbers in fixed-point notation, 
which means that the decimal point is always displayed.

The fixed attribute in the cout statement sets the decimal precision 
of the output to a fixed number of digits after the decimal point, as specified by the setprecision manipulator. 

If you remove the fixed attribute and rerun the program, 
you’ll notice that the output may be represented in scientific notation if 
it has too many zeros after the decimal point or if the number is very large. 

The fixed attribute ensures that the number is represented as a standard decimal number, not in scientific notation. 

This is particularly useful when dealing with monetary values, where we typically want to see exactly two decimal places.

The setprecision() attribute in the cout statement sets the number of decimal places to be displayed for floating-point numbers. 

In this case, setprecision(4) will display four decimal places for the total bill.
    
As you can see, the total bill is now displayed with four decimal places, 
even though the fractional part is still zero. 

This is because the setprecision() attribute has been used to explicitly set the precision to four decimal places.
    
The setprecision() attribute is useful for controlling the precision of floating-point output. 

This can be important for applications where it is important to display precise numerical values.    

The setprecision() attribute in the cout statement sets the number of digits of precision when outputting floating-point values. 

If you change the precision to 4 with the fixed attribute, 
it will output the floating-point number to 4 places after the decimal point. 

As for the showpoint attribute, 
it forces all floating-point output to show a decimal point and trailing zeros, 
even for whole numbers. 

This can be useful for aligning output in columns or ensuring consistent formatting. 

Without showpoint, trailing zeros might be omitted. 

For example, the number 5.00 would be printed as 5 without showpoint, 
but with showpoint, it would be printed as 5.0000 when using setprecision(4).

// This program will read in the quantity of a particular item and its price.
// It will then print out the total price.
// The input will come from the keyboard and the output will go to the screen.

#include iostream
#include iomanip
#include string
using namespace std;

int main() {
    string productName;  // Stores the product name
    int quantity;        // contains the amount of items purchased
    float itemPrice;     // contains the price of each item
    float totalBill;     // contains the total bill

    cout << setprecision(4) << fixed << showpoint;  // formatted output

    // Prompt the user to enter the product name
    cout << "Please input the name of the item: " << endl;
    getline(cin, productName);  // Read the product name into a string

    cout << "Please input the number of items bought:" << endl;
    // Fill in the input statement to bring in the quantity.
    cin >> quantity;

    // Fill in the prompt to ask for the price
    cout << "Please input the price per item: " << endl;

    // Fill in the input statement to bring in the price of each item.
    cin >> itemPrice;

    // Fill in the assignment statement to determine the total bill.
    totalBill = quantity * itemPrice;

    // Display the product name and total bill
    cout << "The item that you bought is " << productName << "." << endl;
    cout << "The total bill is: $" << totalBill << endl;

    return 0;
}

To incorporate the product name into the program, you'll need to include the string library and modify the input and output statements accordingly.

This modified code will prompt the user to enter the product name first, 
and then follow the same process as before for quantity and price.

The output will include the product name along with the total bill.
